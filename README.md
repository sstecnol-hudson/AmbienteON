# AmbienteON

Landing page for **AmbienteON** – a web platform presenting tools, emissions data, and services for environmental management.

## Project Overview
- **Static site** built with plain HTML, CSS and JavaScript.
- Pages: `index.html`, `ferramentas.html`, `emissoes.html`, `calculadora.html`, etc.
- No build step required – just open `index.html` locally or host on a static CDN (Vercel, Netlify, etc.).

## Getting Started
```bash
# Clone the repository (once created on GitHub)
git clone https://github.com/SEU_USUARIO/ambienteon.git
cd ambienteon

# Open the site locally (any browser)
start index.html  # Windows
```

## Development
- Edit the HTML files directly.
- CSS lives in `styles.css` (plus component‑specific files like `admin.css`).
- JavaScript is in `main.js` and additional scripts per page.

## Deployment
The repo can be connected to Vercel or Netlify for automatic deployments. Follow the guide in the repository wiki for detailed steps.
